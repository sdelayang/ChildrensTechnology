Children's Technology

Game

Use the arrows on the keyboard to move the robot within the grid. It is possible to move the robot outside the grid - so be careful.

Use the keyboards 'w', 'a', 's' and 'd' to turn the robot in a specific direction.
w - face the robot to the top/back
a - face the robot to the left
s - face the robot to the bottom/front
d - face the robot to the right
Enter - press enter once the robot reached the gift to start the animation

Reload the page to start a new game.
